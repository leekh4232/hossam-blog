## 프로젝트 폴더 안에 .vscode/tasks.json 파일 만들기

직접 폴더와 파일을 추가하고 아래의 코드를 적용한다.

```json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Git Pull On Startup",
      "type": "shell",
      "command": "git pull",
      "problemMatcher": [],
      "runOptions": {
        "runOn": "folderOpen"
      }
    }
  ]
}
```

이제 `.vscode`폴더가 포함되는 프로젝트를 git으로 열 때, VSCode가 자동으로 Pull을 받게 된다.

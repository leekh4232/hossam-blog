## #02. 구축 시나리오

### 1. 구축할 구성요소

서버 구축은 아래와 같은 시나리오로 진행되었다. 이 포스팅에서는 전반적인 시나리오 제시와 설치된 기본 프로그램들만 소개하고 나머지 서버 구축은 각각 개별 포스팅으로 진행한다.

1. 기본 프로그램 설치
2. 시스템 기본 설정
3. 원격 접속 환경 구성
   - SSH 서버 구성
   - 원격 데스크탑 접속 구성
4. SDK 설치
   - JDK
   - Node.js
   - Python
5. PHP 개발환경 구성 및 SSL 인증서 설정
6. Code Server 구성
7. Jupyter Server 구성
8. MariaDB

### 2. 서비스 포트 구성

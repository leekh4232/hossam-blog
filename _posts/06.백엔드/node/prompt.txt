이 폴더에는 Node.js에 대한 수업자료를 작성하기 위한 준비과정이 포함되어 있어.

"2025-08-28-Node-FileSystem모듈.md" 파일에는 두 번째 단원으로 Node.js를 시작하기에 앞서 필수적으로 익혀야 하는 기본 내장 모듈중에서 FileSystem 모듈에 대한 수업 자료를 작성할거야. 이 파일 이외의 다른 파일은 수정하지 마세요.

수업에 활용할 예제 소스코드는 "02-fs모듈" 폴더에 저장되어 있으며 예제 소스는 파일이름 앞에 순서에 따라 두 자리 일련변호가 적용되어 있어.

md 파일에는 다음과 같은 내용이 먼저 준비되어 있어

```markdown
---
title: {단원제목}
description: {설명글}
categories: {카테고리}
date:   {작성일}
author: {작성자}
image: {썸네일}
tags: {태그}
pin: true
math: true
mermaid: true
---

# {단원제목}
```

이 파일에 현재 작성되어 있는 내용은 그대로 유지하고, 각 실습파일을 익히기 위해 필요한 기본 지식이나 API 소개에 대한 내용을 예제소스 순서에 따라 작성해주세요. 작성되는 내용은 준비된 예제 소스를 설명하기 위한 배경지식, API 소개, 이론, 소스코드 설명 등 입니다.

소스코드 뒷부분에는 소스코드를 설명하는 것도 포함되면 좋습니다.
단, 코드가 너무 간단할 경우에는 소스코드 설명은 생략해도 됩니다.

각 소스 파일별로 `## {번호}. {주제}` 형식으로 제목을 기입하고 그 안에 몇 가지 섹션을 나누어서 실습 코드에 대한 설명을 추가하세요. 섹션의 수는 당신이 임의로 판단해서 최적의 개수를 생성하세요.

소스코드는 설명이 끝난 맨 마지막에 위치하면 됩니다.

예를 들면 아래와 같습니다.

```markdown
## {번호}. {주제}

### 섹션1

섹션1에 대한 내용

### 섹션2

섹션2에 대한 내용

### 실습: {실습주제}

**{실습파일1경로}**

{실습소스1코드}
```

Node를 처음 시작하는 사람이 학습하는 자료이므로 간략하지만 전문적인 내용을 서술해 주세요. 이 자료를 학습하는 사람은 Javascript의 문법은 이미 모두 이해하고 있으므로 문법에 대한 설명은 하지 않아도 됩니다.

완성된 내용은 내가 제시한 파일에 추가해 주세요.

"2025-08-27-Node-기본-내장모듈.md"은 내가 요청한 내용이 반영되어 있는 예시입니다. 하지만 예시보다 더 자세한 수준으로 작성되면 좋습니다.
